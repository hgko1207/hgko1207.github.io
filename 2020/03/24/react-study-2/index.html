2. Project Setup

2.1. Requirements

лЁјм Җ Node.jsк°Җ м„Өм№ҳлҗҳм–ҙ мһҲм–ҙм•ј н•©лӢҲлӢӨ.
Node.js кіөмӢқ лӢӨмҡҙлЎңл“ң нҺҳмқҙм§Җ(https://nodejs.org/ko/download/)м—җм„ң Window InstallerлҘј лӢӨмҡҙлЎңл“ңн•ҳкі  м„Өм№ҳн•©лӢҲлӢӨ.

м—җл””н„°лЎңлҠ” MSм—җм„ң м ңкіөн•ҳлҠ” Visual Studio Code(VS Code)лҘј мӮ¬мҡ©н•©лӢҲлӢӨ. vscodeлҠ” нҒ¬лЎңмҠӨ н”Ңлһ«нҸј м—җл””н„°лЎң лӢӨм–‘н•ң м–ём–ҙлҘј м„ңнҸ¬нҠё н•ҳл©°, IntelliSenseмҷҖ Git кё°лҠҘ, к·ёлҰ¬кі  Extensionмқ„ мқҙмҡ©н•ң нҷ•мһҘ кё°лҠҘмқ„ м ңкіөн•ҳкі  мһҲмҠөлӢҲлӢӨ.
м„Өм№ҳл°©лІ•мқҖ https://code.visualstudio.com/ м—җ м ‘мҶҚн•ҳм—¬ лӢӨмҡҙлЎңл“ң нӣ„ м„Өм№ҳн•ҳл©ҙ лҗ©лӢҲлӢӨ.

React к°ңл°ңм—җ мўӢмқҖ ExtensionлҠ” ESLint, Prettier, vscode-styled-components, Auto Close Tag, React-Native/React/Redux snippets for es6/es7 л“ұ лӢӨм–‘н•ҳкІҢ мһҲм–ҙ м„Өм№ҳн•ҳкі  мӮ¬мҡ©н•ҳл©ҙ лҗ©лӢҲлӢӨ.

2.2. Creating React App

кё°мЎҙм—җлҠ” Webpack, Babel л“ұ н•„мҡ”н•ң лӘЁл“Ҳл“Өмқ„ м§Ғм ‘ м„Өм№ҳн•ҳкі  м„Өм •н•ҳлҠҗлқј мғҒлӢ№н•ң мӢңк°„мқҙ мҶҢмҡ”к°Җ лҗҳм—ҲмҠөлӢҲлӢӨ. 2016л…„м—җ React мһ‘м—… нҷҳкІҪмқ„ лӘ…л №м–ҙ н•ҳлӮҳлЎң м„Өм • н•  мҲҳ мһҲлҠ” кіөмӢқ лҸ„кө¬к°Җ лӮҳмҳӨл©ҙм„ң к°ңл°ңмһҗл“Өкіј мһ…л¬ёмһҗл“Өм—җкІҢ л§ҺмқҖ лҸ„мӣҖмқҙ лҗҳм—ҲмҠөлӢҲлӢӨ.

лЁјм Җ н”„лЎңм қнҠё мғқм„ұмқ„ н•©лӢҲлӢӨ.

м„Өм№ҳк°Җ мҷ„лЈҢлҗҳл©ҙ Visual Studio Code нҺём§‘кё°лЎң н”„лЎңм қнҠёлҘј м—ҪлӢҲлӢӨ.

мІҳмқҢ н”„лЎңм қнҠёк°Җ м„Өм№ҳлҗҳл©ҙ м—¬лҹ¬к°Җм§Җ нҢҢмқјмқҙ мғқм„ұлҗҳлҠ”лҚ° мҙҲкё° м„ёнҢ…мқ„ мң„н•ҙ л¶Ҳн•„мҡ”н•ң нҢҢмқјмқ„ м ңкұ°н•ҳлҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ. src нҸҙлҚ”м—җм„ң App.js, index.js нҢҢмқјмқ„ м ңмҷён•ң нҢҢмқјмқ„ м ңкұ°н•ҳкі  importлҗң мҪ”л“ңлҘј м ңкұ°н•©лӢҲлӢӨ.

3. React к°ңл°ң

3.1. JSX

JSX (JavaScript eXtension)лҠ” мһҗл°”мҠӨнҒ¬лҰҪнҠё м–ём–ҙ л¬ёлІ•мқҳ нҷ•мһҘмһ…лӢҲлӢӨ. мһҗл°”мҠӨнҒ¬лҰҪнҠё м•Ҳм—җм„ң HTML л¬ёлІ•мқ„ мӮ¬мҡ©н•ҙм„ң нҷ”л©ҙмқ„ кө¬м„ұн•  мҲҳ мһҲкІҢ лҸ„мҷҖмЈјлҠ” л¬ёлІ•мңјлЎң, React к°ңл°ңм—җ м—„мІӯлӮң лҸ„мӣҖмқ„ мӨҚлӢҲлӢӨ.

м•„лһҳлҠ” мҠӨнғҖмқјл§Ғмқҳ м—¬лҹ¬к°Җм§Җ л°©лІ•мһ…лӢҲлӢӨ.

  1. class лҢҖмӢ  classNameмқ„ мӮ¬мҡ©н•©лӢҲлӢӨ.
  2. мҠӨнғҖмқј мҶҚм„ұмқҖ мӨ‘кҙ„нҳё ({}) м•Ҳм—җ к°қмІҙ нҳ•нғңлЎң н‘ңмӢңн•ҳл©° лӢЁм–ҙ мӮ¬мқҙмқҳ '-'лҘј м—Ҷм• лҠ” лҢҖмӢ  м№ҙл©ңмјҖмқҙмҠӨ(Camel Case)лҘј мӮ¬мҡ©н•ҙ CSS н”„лЎңнҚјнӢ°лҠ” лӮҳнғҖлғ…лӢҲлӢӨ.
  3. styled-componentsлҠ” лҰ¬м•ЎнҠё CSS-in-JS кҙҖл Ё лқјмқҙлёҢлҹ¬лҰ¬ мӨ‘м—җм„ң к°ҖмһҘ мһҳлӮҳк°ҖлҠ” лқјмқҙлёҢлҹ¬лҰ¬лЎңмҚЁ мһҗл°”мҠӨнҒ¬лҰҪнҠё нҢҢмқј м•Ҳм—җ CSSлҘј мһ‘м„ұн•ҳлҠ” нҳ•нғңмһ…лӢҲлӢӨ.

JSXлҠ” кјӯ м§Җмјңм•ј н•  лӘҮлӘҮ м ңн•ңмқҙ мһҲмҠөлӢҲлӢӨ.

  1. JSXлҘј мӮ¬мҡ©н•ҳлҠ” мҠӨнҒ¬лҰҪнҠё нҢҢмқјмқҖ мғҒлӢЁм—җ React лқјмқҙлёҢлҹ¬лҰ¬лҘј кјӯ л¶Ҳлҹ¬мҷҖм•ј н•©лӢҲлӢӨ.
  2. м—ҙм–ҙ лҶ“мқҖ нғңк·ёлҠ” кјӯ лӢ«м•„м•ј н•©лӢҲлӢӨ.
  3. мөңмғҒмң„ нғңк·ёлҠ” кјӯ 1к°ңм—¬м•ј н•©лӢҲлӢӨ.

3.2. Component кё°л°ҳ кө¬мЎ°

ReactлҠ” Component кё°л°ҳ лқјмқҙлёҢлҹ¬лҰ¬мһ…лӢҲлӢӨ. н•ҳлӮҳмқҳ мҪ”л“ңлЎң мһ‘м„ұн•ҳлҠ” кІғмқҙ м•„лӢҲлқј м—¬лҹ¬ л¶Җл¶„мқ„ л¶„н• н•ҙм„ң л§Ңл“Өкё° л•Ңл¬ём—җ мҪ”л“ңмқҳ мһ¬мӮ¬мҡ©м„ұкіј мң м§ҖліҙмҲҳм„ұмқҙ мҰқк°Җ лҗ©лӢҲлӢӨ.

мң„мҷҖ к°ҷмқҖ htmlмҪ”л“ңк°Җ мһҲлӢӨкі  н•ҙлҙ…мӢңлӢӨ. мқҙлҘј ReactлЎң л§Ңл“ӨкІҢ лҗҳл©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

HeaderлӮҳ Footer, ContentList л“ұмқҖ м»ҙнҸ¬л„ҢнҠёлЎң л§Ңл“Өкі , мқҙлҘј мЎ°лҰҪн•ҙм„ң лЈЁнҠё м»ҙнҸ¬л„ҢнҠёлҘј л§Ңл“ңлҠ” л°©мӢқмһ…лӢҲлӢӨ. м»ҙнҸ¬л„ҢнҠёмқҳ мў…лҘҳлЎңлҠ” нҒҙлһҳмҠӨнҳ•(stateful)кіј н•ЁмҲҳнҳ•(stateless)мңјлЎң лӮҳлҲ„м–ҙм§‘лӢҲлӢӨ.

3.3. Props

propsлһҖ л¶ҖлӘЁ м»ҙнҸ¬л„ҢнҠём—җм„ң мһҗмӢқ м»ҙнҸ¬л„ҢнҠёлЎң м „лӢ¬н•ҙ мЈјлҠ” лҚ°мқҙн„°лҘј л§җн•©лӢҲлӢӨ.
propsлҠ” мқҪкё° м „мҡ© лҚ°мқҙн„°лқјкі  мғқк°Ғн•ҳл©ҙ лҗ©лӢҲлӢӨ. мһҗмӢқ м»ҙнҸ¬л„ҢнҠём—җм„ң м „лӢ¬ л°ӣмқҖ propsлҘј ліҖкІҪмқҙ л¶Ҳк°ҖлҠҘн•ҳкі  propsлҘј м „лӢ¬н•ҙмӨҖ мөңмғҒмң„ л¶ҖлӘЁ м»ҙнҸ¬л„ҢнҠёл§Ң propsлҘј ліҖкІҪн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

3.4. State

stateлҠ” лҸҷм Ғмқё лҚ°мқҙн„°лҘј лӢӨлЈ° л•Ң мӮ¬мҡ©н•©лӢҲлӢӨ. мӮ¬мҡ©мһҗмҷҖмқҳ мғҒнҳёмһ‘мҡ©мқ„ нҶөн•ҙ лҚ°мқҙн„°лҘј лҸҷм ҒмңјлЎң ліҖкІҪмқ„ н•ҙм•ј н•  л•Ң мӮ¬мҡ©н•©лӢҲлӢӨ.
stateлҠ” нҒҙлһҳмҠӨнҳ• м»ҙнҸ¬л„ҢнҠём—җм„ңл§Ң мӮ¬мҡ©н•  мҲҳ мһҲлҠ”лҚ° к°Ғк°Ғмқҳ stateлҠ” лҸ…лҰҪм Ғмқҙлқј
лӢӨлҘё м»ҙнҸ¬л„ҢнҠёмқҳ м§Ғм ‘м Ғмқё м ‘к·јмқҖ л¶Ҳк°ҖлҠҘн•ҳлӢӨ.

3.5 LifeCycle API

LifeCycle APIлҠ” м»ҙнҸ¬л„ҢнҠёк°Җ DOM мң„м—җ мғқм„ұлҗҳкё° м „ нӣ„ л°Ҹ лҚ°мқҙн„°к°Җ ліҖкІҪлҗҳм–ҙ мғҒнғңлҘј м—…лҚ°мқҙнҠён•ҳкё° м „ нӣ„лЎң мӢӨн–үлҗҳлҠ” л©”мҶҢл“ңл“Ө мһ…лӢҲлӢӨ. мһҗмӣҗлӮӯл№„лҘј мӨ„мқҙкё° мң„н•ҳм—¬ мҪ”л“ңлҘј мөңм Ғнҷ” н•  л•Ң мӮ¬мҡ©лҗ©лӢҲлӢӨ. л§Һмқҙ мӮ¬мҡ©н•ҳлҠ” LifeCycle л©”мҶҢл“ңл“ӨмқҖ constructor, componentDidMount, render, shouldComponentUpdate мһ…лӢҲлӢӨ.
