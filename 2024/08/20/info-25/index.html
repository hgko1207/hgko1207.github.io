Swagger vs. Postman

2024-08-20
API, Api Development, IT, Postman, Swagger, кё°мҲ 
Symbols count in article: 604 Reading time: 3 mins.

м—ӯлҸҷм Ғмқё API к°ңл°ң мҳҒм—ӯм—җм„ң лҸ„кө¬лҠ” н”„лЎңм„ёмҠӨлҘј к°„мҶҢнҷ”н•ҳкі  нҳ‘м—…мқ„ к°•нҷ”н•ҳл©° м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ мӣҗнҷңн•ң мһ‘лҸҷмқ„ ліҙмһҘн•ҳлҠ” лҚ° мӨ‘м¶”м Ғмқё м—ӯн• мқ„ н•©лӢҲлӢӨ. мқҙ мҳҒм—ӯмқҳ лҢҖн‘ңм Ғмқё л‘җ к°Җм§Җ лҸ„кө¬лҠ” к°Ғк°Ғ кі мң н•ң кё°лҠҘкіј мӮ¬мҡ© мӮ¬лЎҖлҘј м ңкіөн•ҳлҠ” SwaggerмҷҖ Postmanмһ…лӢҲлӢӨ. мқҙ кёҖм—җм„ңлҠ” SwaggerмҷҖ Postmanмқҳ к°•м җкіј нҷңмҡ© мӮ¬лЎҖлҘј мһҗм„ёнһҲ мӮҙнҺҙліҙкі , нҠ№м • мҡ”кө¬мӮ¬н•ӯм—җ л”°лқј м •ліҙм—җ мһ…к°Ғн•ң кІ°м •мқ„ лӮҙлҰҙ мҲҳ мһҲлҸ„лЎқ лҸ„мҷҖл“ңлҰҪлӢҲлӢӨ.

SwaggerлһҖ л¬ҙм—Үмқёк°Җмҡ”?

нҳ„мһ¬ OpenAPI мӮ¬м–‘мңјлЎң м•Ңл Өм§„ SwaggerлҠ” к°ңл°ңмһҗк°Җ RESTful APIлҘј м„Өкі„, л¬ём„ңнҷ” л°Ҹ н…ҢмҠӨнҠён•  мҲҳ мһҲлҠ” мҳӨн”Ҳ мҶҢмҠӨ н”„л Ҳмһ„мӣҢнҒ¬мһ…лӢҲлӢӨ. REST APIлҘј м„ӨлӘ…н•ҳлҠ” н‘ңмӨҖнҷ”лҗң л°©лІ•мқ„ м ңкіөн•ҳм—¬ мӮ¬лһҢкіј м»ҙн“Ён„° лӘЁл‘җ мҶҢмҠӨ мҪ”л“ңм—җ м•Ўм„ёмҠӨн•ҳм§Җ м•Ҡкі лҸ„ м„ңл№„мҠӨмқҳ кё°лҠҘмқ„ мқҙн•ҙн•  мҲҳ мһҲлҸ„лЎқ н•ҙмӨҚлӢҲлӢӨ.

Swaggerмқҳ мӮ¬мҡ© мӮ¬лЎҖ

  • API л¬ём„ң: SwaggerлҠ” нҸ¬кҙ„м Ғмқё лҢҖнҷ”нҳ• API л¬ём„ңлҘј мғқм„ұн•ҳлҠ” лҚ° нғҒмӣ”н•©лӢҲлӢӨ. кё°кі„к°Җ мқҪмқ„ мҲҳ мһҲлҠ” нҳ•мӢқмңјлЎң APIлҘј м„ӨлӘ…н•ЁмңјлЎңмҚЁ к°ңл°ңмһҗк°Җ APIлҘј ліҙлӢӨ нҡЁмңЁм ҒмңјлЎң мқҙн•ҙн•ҳкі  мӮ¬мҡ©н•  мҲҳ мһҲлҸ„лЎқ м§Җмӣҗн•©лӢҲлӢӨ.
  • API л””мһҗмқё: SwaggerлҘј мӮ¬мҡ©н•ҳл©ҙ Swagger м—җл””н„°лҘј мӮ¬мҡ©н•ҳм—¬ мӢңк°Ғм ҒмңјлЎң APIлҘј л””мһҗмқён•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҹ¬н•ң мӢңк°Ғм Ғ м ‘к·ј л°©мӢқмқҖ нҢҖмӣҗ к°„мқҳ нҳ‘м—…мқ„ мҙүм§„н•ҳкі  н‘ңмӨҖнҷ”лҗң API м„Өкі„ н”„лЎңм„ёмҠӨлҘј ліҙмһҘн•©лӢҲлӢӨ.
  • мҪ”л“ң мғқм„ұ: SwaggerлҠ” лӢӨм–‘н•ң н”„лЎңк·ёлһҳл°Қ м–ём–ҙлЎң м„ңлІ„ Stubкіј нҒҙлқјмқҙм–ёнҠё SDKлҘј мғқм„ұн•  мҲҳ мһҲм–ҙ API кё°лҠҘмқ„ кө¬нҳ„н•ҳлҠ” лҚ° н•„мҡ”н•ң мҲҳмһ‘м—…мқ„ мӨ„м—¬мӨҚлӢҲлӢӨ.
  • н…ҢмҠӨнҠё: SwaggerлҠ” Swagger UIмҷҖ к°ҷмқҖ лҸ„кө¬мҷҖмқҳ нҶөн•©мқ„ нҶөн•ҙ API н…ҢмҠӨнҠёлҘј мҡ©мқҙн•ҳкІҢ н•©лӢҲлӢӨ. к°ңл°ңмһҗлҠ” л¬ём„ңм—җм„ң м§Ғм ‘ API мҡ”мІӯмқ„ мӢӨн–үн•  мҲҳ мһҲмңјлҜҖлЎң н…ҢмҠӨнҠё л°Ҹ л””лІ„к№…м—җ мң мҡ©н•ң лҸ„кө¬мһ…лӢҲлӢӨ.

PostmanмқҙлһҖ л¬ҙм—Үмқёк°Җмҡ”?

PostmanмқҖ APIлҘј м„Өкі„, н…ҢмҠӨнҠё л°Ҹ кҙҖлҰ¬н•ҳлҠ” кіјм •мқ„ к°„мҶҢнҷ”н•ҳлҠ” API к°ңл°ңмҡ© нҳ‘м—… н”Ңлһ«нҸјмңјлЎң л„җлҰ¬ мӮ¬мҡ©лҗҳкі  мһҲмҠөлӢҲлӢӨ. HTTP мҡ”мІӯмқ„ мғқм„ұн•ҳкі  м „мҶЎн•  мҲҳ мһҲлҠ” мӮ¬мҡ©мһҗ м№ңнҷ”м Ғмқё мқён„°нҺҳмқҙмҠӨлҘј м ңкіөн•  лҝҗл§Ң м•„лӢҲлқј APIмқҳ н…ҢмҠӨнҠё л°Ҹ лӘЁлӢҲн„°л§Ғмқ„ мһҗлҸҷнҷ”н•©лӢҲлӢӨ.

Postman мӮ¬мҡ© мӮ¬лЎҖ

  • API н…ҢмҠӨнҠё: PostmanмқҖ лӢӨлӘ©м Ғ API н…ҢмҠӨнҠё лҸ„кө¬лЎңм„ң л№ӣмқ„ л°ңн•©лӢҲлӢӨ. к°ңл°ңмһҗлҠ” API мҡ”мІӯмқ„ мғқм„ұ л°Ҹ мӢӨн–үн•ҳкі , мһҗлҸҷнҷ”лҗң н…ҢмҠӨнҠё мҠӨмң„нҠёлҘј м„Өм •н•ҳкі , нҡҢк·Җ н…ҢмҠӨнҠёлҘј мҲҳн–үн•ҳм—¬ APIмқҳ м•Ҳм •м„ұмқ„ ліҙмһҘн•  мҲҳ мһҲмҠөлӢҲлӢӨ.
  • API м„Өкі„ л°Ҹ н”„лЎңнҶ нғҖмқҙн•‘: Postmanмқ„ мӮ¬мҡ©н•ҳл©ҙ н”Ңлһ«нҸј лӮҙм—җм„ң м§Ғм ‘ APIлҘј м„Өкі„н•ҳкі  н”„лЎңнҶ нғҖмқҙн•‘н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙ кё°лҠҘмқҖ л№ лҘё л°ҳліөкіј н”јл“ңл°ұ лЈЁн”„лҘј к°ҖлҠҘн•ҳкІҢ н•ҳм—¬ к°ңл°ң н”„лЎңм„ёмҠӨлҘј к°ҖмҶҚнҷ”н•©лӢҲлӢӨ.
  • нҳ‘м—…: PostmanмқҖ кіөмң  мһ‘м—… кіөк°„, API лІ„м „ кҙҖлҰ¬, мҡ”мІӯ л°Ҹ м»¬л үм…ҳм—җ лҢ“кёҖмқ„ лӢ¬ мҲҳ мһҲлҠ” кё°лҠҘмқ„ м ңкіөн•ҳм—¬ нҢҖмӣҗ к°„мқҳ нҳ‘м—…мқ„ мҙүм§„н•©лӢҲлӢӨ.
  • лӘЁлӢҲн„°л§Ғ л°Ҹ мһҗлҸҷнҷ”: Postmanмқ„ мӮ¬мҡ©н•ҳл©ҙ API н…ҢмҠӨнҠё м»¬л үм…ҳмқ„ мһҗлҸҷмңјлЎң мӢӨн–үн•ҳкі  мҳҲм•Ҫн•ҳлҠ” лӘЁлӢҲн„°лҘј л§Ңл“Ө мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҠ” м§ҖмҶҚм Ғмқё нҶөн•©кіј APIмқҳ мғҒнғңлҘј лӘЁлӢҲн„°л§Ғн•ҳлҠ” лҚ° мң мҡ©н•©лӢҲлӢӨ.

SwaggerмҷҖ Postman: мҳ¬л°”лҘё лҸ„кө¬ м„ нғқн•ҳкё°

SwaggerлҘј м„ нғқн•ҙм•ј н•ҳлҠ” кІҪмҡ°

  • API м„Өкі„ мҡ°м„  м ‘к·ј л°©мӢқ: к°ңл°ң н”„лЎңм„ёмҠӨк°Җ кө¬нҳ„ м „м—җ API мӮ¬м–‘мқ„ м •мқҳн•ҳлҠ” API м„Өкі„ мҡ°м„  м ‘к·ј л°©мӢқмқ„ л”°лҘҙлҠ” кІҪмҡ° Swaggerк°Җ нғҒмӣ”н•ң м„ нғқмһ…лӢҲлӢӨ.
  • нҸ¬кҙ„м Ғмқё л¬ём„ң: мғҒм„ён•ң лҢҖнҷ”нҳ• API л¬ём„ңлҘј мһ‘м„ұн•ҳлҠ” кІғмқҙ мҡ°м„ мҲңмң„лқјл©ҙ Swaggerмқҳ кҙ‘лІ”мң„н•ң л¬ём„ң мғқм„ұ кё°лҠҘмқҙ мң мҡ©н•ң лҸ„кө¬к°Җ лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.
  • мҪ”л“ң мғқм„ұ: API мӮ¬м–‘м—җ л”°лқј м„ңлІ„ StubмқҙлӮҳ нҒҙлқјмқҙм–ёнҠё SDKлҘј мһҗлҸҷмңјлЎң мғқм„ұн•ҙм•ј н•ҳлҠ” кІҪмҡ° SwaggerлҠ” к°•л Ҙн•ң мҪ”л“ң мғқм„ұ кё°лҠҘмқ„ м ңкіөн•©лӢҲлӢӨ.

Postmanмқ„ м„ нғқн•ҙм•ј н•ҳлҠ” кІҪмҡ°

  • API н…ҢмҠӨнҠё мӨ‘мӢ¬: нҠ№нһҲ мһҗлҸҷнҷ”лҗң н…ҢмҠӨнҠё, н…ҢмҠӨнҠё мҠӨмң„нҠё мғқм„ұ, лӘЁлӢҲн„°л§Ғкіј к°ҷмқҖ кё°лҠҘмқ„ нҶөн•ҙ API н…ҢмҠӨнҠём—җ мӨ‘м җмқ„ л‘җлҠ” кІҪмҡ°, PostmanмқҖ мқҙлҹ¬н•ң мһ‘м—…м—җ л§һм¶Өнҷ”лҗң м„ңл№„мҠӨмһ…лӢҲлӢӨ.
  • н”„лЎңнҶ нғҖмқҙн•‘ л°Ҹ л°ҳліө: лҸ„кө¬ лӮҙм—җм„ң м§Ғм ‘ APIлҘј л№ лҘҙкІҢ н”„лЎңнҶ нғҖмқҙн•‘н•ҳкі  л°ҳліөн•  мҲҳ мһҲлҠ” н”Ңлһ«нҸјмқ„ м„ нҳён•ңлӢӨл©ҙ PostmanмқҖ мқҙлҹ¬н•ң нҷңлҸҷмқ„ мң„н•ң м§ҒкҙҖм Ғмқё нҷҳкІҪмқ„ м ңкіөн•©лӢҲлӢӨ.
  • нҳ‘м—… к°ңл°ң: API к°ңл°ң мһ‘м—… кіөк°„мқҳ нҳ‘м—…кіј кіөмң к°Җ нҢҖм—җ л§Өмҡ° мӨ‘мҡ”н•ҳлӢӨл©ҙ Postmanмқҳ нҳ‘м—… кё°лҠҘмқҙ мӣҗнҷңн•ң нҷҳкІҪмқ„ м ңкіөн•©лӢҲлӢӨ.

кІ°лЎ 

API к°ңл°ңмқҳ м—ӯлҸҷм Ғмқё нҷҳкІҪм—җм„ң SwaggerмҷҖ PostmanмқҖ лӘЁл‘җ к°ңл°ң мҲҳлӘ… мЈјкё° м „л°ҳм—җ кұёміҗ нҠ№м • мҡ”кө¬ мӮ¬н•ӯмқ„ н•ҙкІ°н•ҳл©ҙм„ң лҸ…нҠ№н•ң нӢҲмғҲ мӢңмһҘмқ„ к°ңмІҷн•ҙ мҷ”мҠөлӢҲлӢӨ. SwaggerлҠ” API л””мһҗмқё, л¬ём„ңнҷ” л°Ҹ мҪ”л“ң мғқм„ұм—җ нғҒмӣ”н•ҳм§Җл§Ң PostmanмқҖ API н…ҢмҠӨнҠё, нҳ‘м—… л°Ҹ н”„лЎңнҶ нғҖмқҙн•‘ мҳҒм—ӯм—җм„ң л№ӣмқ„ л°ңн•©лӢҲлӢӨ. SwaggerмҷҖ Postman мӨ‘ м–ҙл–Ө кІғмқ„ м„ нғқн• м§ҖлҠ” к¶Ғк·№м ҒмңјлЎң н”„лЎңм қнҠё мҡ”кө¬ мӮ¬н•ӯ, нҢҖ м„ нҳёлҸ„, н–ҘмғҒлҗң нҡЁмңЁм„ұкіј кё°лҠҘмқ„ мӣҗн•ҳлҠ” API к°ңл°ң н”„лЎңм„ёмҠӨмқҳ нҠ№м • лӢЁкі„м—җ л”°лқј лӢ¬лқјм§‘лӢҲлӢӨ.

SwaggerмҷҖ Postmanмқҳ к°•м җмқ„ нҷңмҡ©н•ҳм—¬ API к°ңл°ң лҸ„кө¬ нӮӨнҠём—җ нҶөн•©н•ҳм—¬ нҸ¬кҙ„м Ғмқҙкі  к°„мҶҢнҷ”лҗң к°ңл°ң нҷҳкІҪмқ„ кө¬м¶•н•ҳм„ёмҡ”. APIлҘј м„Өкі„н•ҳкұ°лӮҳ л¬ём„ңнҷ”н•ҳкұ°лӮҳ кё°лҠҘмқ„ м—„кІ©н•ҳкІҢ н…ҢмҠӨнҠён•  л•Ң SwaggerмҷҖ Postmanмқҳ мӢңл„Ҳм§Җ нҡЁкіјлЎң к°ңл°ңмһҗлҠ” мөңмӢ  API к°ңл°ңмқҳ ліөмһЎн•ң нҷҳкІҪмқ„ мһҗмӢ  мһҲкІҢ нғҗмғүн•  мҲҳ мһҲмҠөлӢҲлӢӨ.
