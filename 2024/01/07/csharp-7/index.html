C#м—җм„ң Stringкіј stringмқҳ м°Ёмқҙм җ мқҙн•ҙн•ҳкё°

2024-01-07
C#, Programming, String, string
Symbols count in article: 466 Reading time: 2 mins.

C#мқҖ к°•л Ҙн•ң нҳ•мӢқмқҳ н”„лЎңк·ёлһҳл°Қ м–ём–ҙмһ…лӢҲлӢӨ. нҡЁкіјм Ғмқё C# мҪ”л“ңлҘј мһ‘м„ұн•ҳл Өл©ҙ лҚ°мқҙн„° нҳ•мӢқмқ„ м•„лҠ” кІғмқҙ н•„мҲҳм Ғмһ…лӢҲлӢӨ. C#м—җм„ң к°ҖмһҘ мһҗмЈј мӮ¬мҡ©лҗҳлҠ” лҚ°мқҙн„° мң нҳ• мӨ‘ н•ҳлӮҳлҠ” л¬ёмһҗм—ҙ мң нҳ•мңјлЎң String, stringлЎң м„ м–ён•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлЎң мқён•ҙ м•Ҫк°„мқҳ нҳјлҸҷмқҙ л°ңмғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ. Stringкіј stringмқҳ м°Ёмқҙм җмқҖ л¬ҙм—Үмқҙл©°, м–ём ң к°Ғк°Ғмқ„ мӮ¬мҡ©н•ҙм•ј н• к№Ңмҡ”?

мқҙ кёҖм—җм„ңлҠ” мқҙ л‘җ к°Җм§Җ мң нҳ•мқ„ мһҗм„ёнһҲ мӮҙнҺҙліҙкі  мң мӮ¬м җкіј м°Ёмқҙм җмқ„ мқҙн•ҙн•ҳкі  м–ҙлҠҗ мӘҪмқ„ лҚ” м„ нҳён•  мҲҳ мһҲлҠ” лӘҮ к°Җм§Җ мӢңлӮҳлҰ¬мҳӨлҘј мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ.

Stringкіј stringмқҖ л¬ҙм—Үмһ…лӢҲк№Ң?

String

String нҒҙлһҳмҠӨлҠ” мӢңмҠӨн…ң л„Өмһ„мҠӨнҺҳмқҙмҠӨ(System.String)мқҳ мқјл¶Җмһ…лӢҲлӢӨ. Substring, Concat, Replace л“ұкіј к°ҷмқҖ л¬ёмһҗм—ҙ мЎ°мһ‘мқ„ мң„н•ң лӢӨм–‘н•ң л©”мҶҢл“ңлҘј м ңкіөн•ҳлҠ” нҒҙлһҳмҠӨмһ…лӢҲлӢӨ. лӢӨмқҢкіј к°ҷмқҙ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

string

л°ҳл©ҙм—җ stringмқҖ C# м–ём–ҙмқҳ Stringмқҳ лі„м№ӯмһ…лӢҲлӢӨ. л¬ёмһҗм—ҙ ліҖмҲҳлҘј м„ м–ён•ҳлҠ” лҚ” мқҪкё° мү¬мҡҙ л°©лІ•мқҙл©° кө¬л¬ёмғҒ лҚ” к°„лӢЁн•©лӢҲлӢӨ. stringмқҖ лӢЁмҲңнһҲ System.Stringмқҳ м•Ҫм–ҙмһ…лӢҲлӢӨ.

мң мӮ¬м җ

  1. мғҒнҳё көҗнҷҳ к°ҖлҠҘ: Stringкіј stringмқ„ л¬ём ңм—Ҷмқҙ л°”кҝ”м„ң мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.
  1. лҸҷмқјн•ң л©”м„ңл“ң: stringмқҖ System.Stringмқҳ лі„м№ӯмқј лҝҗмқҙлҜҖлЎң л‘ҳ лӘЁл‘җм—җ лҸҷмқјн•ң л©”м„ңл“ңлҘј мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

м°Ёмқҙм җ

кё°лҠҘм ҒмңјлЎңлҠ” лҸҷмқјн•ҳм§Җл§Ң м–ём ң м–ҙлҠҗ мӘҪмқ„ мӮ¬мҡ©н•ҙм•ј н•ҳлҠ”м§Җм—җ лҢҖн•ң лӘҮ к°Җм§Җ лӘЁлІ” мӮ¬лЎҖк°Җ мһҲмҠөлӢҲлӢӨ.

  1. к°ҖлҸ…м„ұ: л§ҺмқҖ к°ңл°ңмһҗл“ӨмқҖ string мӮ¬мҡ©н•ҳлҠ” кІғмқҙ int, bool, doubleкіј к°ҷмқҖ лӢӨлҘё лӢЁмҲңн•ң мң нҳ•кіј мһҳ м–ҙмҡёлҰ¬кё° л•Ңл¬ём—җ к°ҖлҸ…м„ұмқҙ лҚ” лҶ’лӢӨкі  мғқк°Ғн•©лӢҲлӢӨ.

  2. Namespacing: using System;мқ„ мӮ¬мҡ©н•ҳм—¬ м„ м–ён•ң кІҪмҡ° л¬ём ңм—Ҷмқҙ Stringмқ„ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. к·ёл Үм§Җ м•Ҡмңјл©ҙ System.StringмңјлЎң мӮ¬мҡ©н•ҙм•ј н•©лӢҲлӢӨ.

  3. н‘ңмӨҖнҷ”: лҢҖл¶Җл¶„мқҳ C# мҪ”л”© н‘ңмӨҖм—җм„ңлҠ” м§Җм—ӯ ліҖмҲҳ л°Ҹ л©”м„ңл“ң л§Өк°ң ліҖмҲҳм—җ лҢҖн•ҙ stringліҙлӢӨ Stringмқ„ мӮ¬мҡ©н•  кІғмқ„ к¶ҢмһҘн•©лӢҲлӢӨ. л°ҳнҷҳ(return) мң нҳ•мқҳ кІҪмҡ° л‘ҳ лӢӨ мӮ¬мҡ©н•  мҲҳ мһҲм§Җл§Ң мқјл°ҳм ҒмңјлЎң мқјкҙҖм„ұмқ„ мң м§Җн•ҳлҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ.

Stringкіј string мӮ¬мҡ© мӢңкё°

String

  1. м •м Ғ л©”м„ңл“ң(Static Method)лЎң мһ‘м—…н•ҳкё°: String нҒҙлһҳмҠӨмқҳ м •м Ғ л©”м„ңл“ңлҘј мӮ¬мҡ©н•  л•ҢлҠ” Stringмқ„ мӮ¬мҡ©н•ҳлҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ.
  1. м •к·ңнҷ”лҗң мқҙлҰ„ н•„мҡ”: л„Өмһ„мҠӨнҺҳмқҙмҠӨк°Җ л¬ём ңк°Җ лҗҳлҠ” нҷҳкІҪм—җм„ң мһ‘м—…н•ҳлҠ” кІҪмҡ°(л§Өмҡ° л“ңл¬јм§Җл§Ң) System.Stringмқ„ м§Җм •н•ҙм•ј н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

string

  1. м§Җм—ӯ ліҖмҲҳ л°Ҹ л§Өк°ңліҖмҲҳ: лЎңм»¬ ліҖмҲҳ м„ м–ёкіј л©”м„ңл“ң л§Өк°ңліҖмҲҳ мң нҳ•м—җлҠ” stringмқ„ мӮ¬мҡ©н•ҳлҠ” кІғмқҙ мқјл°ҳм Ғмһ…лӢҲлӢӨ.
  1. к°ҖлҸ…м„ұ: stringмқ„ мӮ¬мҡ©н•ҳл©ҙ лӢӨлҘё кё°ліё м ңкіө мң нҳ•(int, bool л“ұ)мқ„ мӮ¬мҡ©н•  л•ҢмҷҖ мҪ”л“ңмқҳ мқјкҙҖм„ұмқ„ мң м§Җн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

кІ°лЎ 

C#мқҳ Stringкіј string мң нҳ•мқҖ кё°лҠҘм ҒмңјлЎң лҸҷмқјн•ҳм§Җл§Ң л‘ҳ мӨ‘ н•ҳлӮҳлҘј м„ нғқн•  л•Ң кі л Өн•ҙм•ј н•  лӘҮ к°Җм§Җ лҜёл¬ҳн•ң лӘЁлІ” мӮ¬лЎҖк°Җ мһҲмҠөлӢҲлӢӨ. лҢҖл¶Җл¶„мқҳ к°ңл°ңмһҗлҠ” лЎңм»¬ ліҖмҲҳ л°Ҹ л§Өк°ңліҖмҲҳм—җлҠ” stringмқ„ мӮ¬мҡ©н•ҳкі  м •м Ғ л©”м„ңл“ң нҳём¶ңм—җлҠ” Stringмқ„ мӮ¬мҡ©н•ҳл©ҙ мҪ”л“ңлҘј к°ҖлҸ…м„ұ мһҲкі  мқјкҙҖм„ұ мһҲкІҢ л§Ңл“Ө мҲҳ мһҲмҠөлӢҲлӢӨ.
