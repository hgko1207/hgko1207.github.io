Angular н”„лЎңм қнҠёлҘј н•ң лІҲлҸ„ н•ҙліё м Ғмқҙ м—ҶлҠ” кІҪмҡ° мғҲ н”„лЎңм қнҠёлҘј л§Ңл“ңлҠ” л°©лІ•мқ„ ліҙм—¬ л“ңлҰ¬кІ мҠөлӢҲлӢӨ. н•ҳм§Җл§Ң н•ҙлӢ№ л¶Җл¶„мқ„ кұҙл„Ҳлӣ°кі  OpenLayersлҘј кё°мЎҙ н”„лЎңм қнҠём—җ м¶”к°Җн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мғҲ Angular н”„лЎңм қнҠё л§Ңл“Өкё°

мқҙ мӢңм җм—җм„ң мІ« лІҲм§ё Angular н”„лЎңм қнҠёлҘј м„Өм •н•ҳлҠ” л°©лІ•мқ„ ліҙм—¬ л“ңлҰ¬кІ мҠөлӢҲлӢӨ. лӘЁл“  м»ҙн“Ён„° мҡҙмҳҒ мІҙм ңлҘј мӮ¬мҡ©н•  мҲҳ мһҲмңјл©° AngularлҠ” macOS, Linux л°Ҹ Windowsм—җм„ң мһ‘лҸҷн•©лӢҲлӢӨ. мқҙкІғмқҖ м§ҖкёҲ мӮ¬мҡ©н•ҳлҠ” м„Өм •мһ…лӢҲлӢӨ.

лЁјм Җ AngularлҘј мІҳмқҢ мӮ¬мҡ©н•ҳлҠ” кІҪмҡ° лЎңм»¬ нҷҳкІҪмқ„ м„Өм •н•ҙм•ј н•©лӢҲлӢӨ. Node.js мҷҖ npmмқҙ н•„мҡ”н•©лӢҲлӢӨ.

м„Өм№ҳлҗң лІ„м „мқ„ нҷ•мқён•ҳл Өл©ҙ лӢӨмқҢ лӘ…л №мқ„ мӢӨн–үн•ҳмӢӯмӢңмҳӨ.

мһҗ, мғҲ н”„лЎңм қнҠёлҘј л§Ңл“Өм–ҙ лҙ…мӢңлӢӨ.

command line interface(н„°лҜёл„җ, лӘ…л № н”„лЎ¬н”„нҠё)лҘј м—ҙкі  мғҲ Angular н”„лЎңм қнҠём—җ мӮ¬мҡ©н•ҳл ӨлҠ” л””л үнҶ лҰ¬лЎң мқҙлҸҷн•©лӢҲлӢӨ. Angular CLIлҘј м„Өм№ҳн•ҳл Өл©ҙ лӢӨмқҢмқ„ мӢӨн–үн•©лӢҲлӢӨ.

Angular н”„лЎңм қнҠём—җ OpenLayers м¶”к°Җ
